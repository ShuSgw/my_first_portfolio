Vision Web

Summary

Vision Web is a website for one of assignment from project management class at BCIT. I was the only programmer at the team and creating almost 20 pages website by myself within a week. Skills I used are HTML, Sass, jQuery, node package manager and one of its modules called browsefy. The main purpose is to practice basic web development skills especially to get familier with modern web development tools, such as node package manager.
