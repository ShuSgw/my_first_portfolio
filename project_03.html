Original WordPress Theme

Summary

Creating WordPress theme from scratch was my first challenge, so I spent one week to complete it. The purple of the project is to show my design and coding skills in HTML, CSS, JavaScript, WordPress, and PHP. I created the wireframe first, and followed the design with F Shaped pattern which is the most common user scanning patterns when it comes to blocks of contents in western countries. Additionally, the colour scheme I chose is the same as one I chose to my portfolio for the purpose of branding by myself. In the near future, I will publish the WordPress Theme in a place where users can download as free and allow them to use.
