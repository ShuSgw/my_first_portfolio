Sharable Birhday Card

Summary

"Sharable Birthday Card" is a single webpage base application created by HTML, CSS, and Javascript with jQuery platform. All images I used are SVG data and I control them by CSS animation and jQuery. There are three basic interactive functions on this website. Users can control the position of drinks with drag-and-drop. Also, when users tap a balloon, it goes away. Finally, the note illustration is also interactive. When users tap or click it, the jQuery connect to JSON based iTunes database by Ajax which can perform an asynchronous HTTP request. When the Ajax bring the iTunes Date back, JavaScript Audio API receives the information and play music randomly.

  • HTML5
  • CSS3
  • JavaScript
  • jQuery
  • Ajax
  • iTunes API
