Pet Finder

Summary

Pet finder is a website to find images of dogs based on their names from the database made by JSON. Also, I used bootstrap which is a famous CSS framework to style properly. The purpose of the project is to show my skills of Fetch API and Bootstrap. To receiving the images, I used Fetch API which is the newest asynchronous technology to communicate with a server.

  • HTML5
  • CSS3
  • JavaScript
  • Fetch
  • Dog API
  • BootStrap
