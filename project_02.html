Pet Finder

Summary

Pet finder is a website to find images of dogs based on their names from the database made by JSON. Also I used bootstrap which is a famous CSS framework to design properly. The process of receiving image data is making selections with Javascript and pass the values when the selection is selected into server based on JSON. When Javascript send the value into data, I use Feach API which is the newest method to do asynchronous process.

  • HTML5
  • CSS3
  • JavaScript
  • Fetch
  • Dog API
  • BootStrap
